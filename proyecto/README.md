# RecetaCreativa

Estructura principal del proyecto:

- `*.html`: páginas públicas del sitio.
- `assets/css/`: hojas de estilo.
- `assets/js/`: scripts del navegador.
- `auth/`: endpoints PHP usados por `fetch`.
- `config/`: conexión a la base de datos.
- `partials/`: fragmentos reutilizables de HTML.
- `media/`: imágenes de recetas y perfil.
- `docs/`: documentos del proyecto.
- `legacy/`: PHP duplicado o antiguo que ya no usan las páginas.
